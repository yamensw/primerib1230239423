# Prime Rig — WebAR Showroom

This is a static portfolio/showroom site.

## Run locally
Just open `index.html` in a browser (or use a local server).

## Deploy on GitHub Pages
1. Push this folder to a GitHub repo (root contains `index.html`).
2. Repo Settings → Pages → Deploy from branch → `main` → `/root`.
3. Your site will publish automatically.

Assets are in `/assets`.
